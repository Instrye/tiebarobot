# tiebarobot#
*需要Redis支持*

*需要cURL支持*

这是一个可以自动根据规则删除百度贴吧帖子的“机器人”

###适用范围###
- 应对爆吧
- 应对广告
- 应对没有特定格式

###效果###
![sp170221_114756.png](https://ooo.0o0.ooo/2017/02/21/58abb8874ed2c.png)

###食用方法###

-利用`cron`定时运行 `index.php`

-利用监控定时请求 `index.php`
## 目前##
1. 标题内必须包含
2. 内容中不能包含
- 支持删除贴内回复


##规划##

- 支持自动封禁
- 支持图片识别
- 支持语义分析
